## Available commands

### /clarity-rephrase

Rephrase a message before sending it. Clarity analyzes your text and suggests improvements.

```
/clarity-rephrase [your message]
```

**Example:**

```
/clarity-rephrase hey can u fix this bug its blocking me
```

Clarity suggests:

> Hi, could you help fix this bug? It's currently blocking my work.

Click **Send** to post the improved message or **Didn't like it** to dismiss.

---

### /clarity-settings

Open the settings modal to manage your coaching preferences.

```
/clarity-settings
```

In the settings modal you can:

- Enable or disable coaching flags
- Select channels for auto coaching
- Edit or create custom flags (Pro plan)
- Access billing (workspace admin only)

---

### /clarity-help

Display a list of available commands and quick tips.

```
/clarity-help
```

---

### /clarity-status

Check whether Clarity is active in the current channel.

```
/clarity-status
```

Returns information about:

- Whether auto coaching is enabled
- Which coaching flags are active
- Your usage for the current month

---

### /clarity-feedback

Send feedback directly to the Clarity team.

```
/clarity-feedback [your feedback]
```

**Example:**

```
/clarity-feedback Love the app! Would be great to have a dark mode.
```

Your feedback helps us improve Clarity for everyone.

## Tips

<Tip>
All commands work in any channel or direct message. Responses are only visible to you.
</Tip>

<Note>
Commands count toward your monthly usage limits. Check your plan details in [Billing](/billing).
</Note>
